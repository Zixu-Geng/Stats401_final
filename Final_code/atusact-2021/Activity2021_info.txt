
The data file atusact_2021.dat (2021 Activity file) contains 31 variables and 164,581 observations.  

For more information about this file, see the ATUS 2021 Interview Data Dictionary, available online at:  https://www.bls.gov/tus/atusintcodebk21.pdf.

In addition to the data file, three computer programs are contained in atusact-2021.zip.  If you have SPSS, SAS, or Stata software, you may use these programs to read the data into your statistical software package.  The programs include variable and value labels for most variables.  For information about unlabeled values, please refer to the data dictionary.

The file "atusact_2021.sps" reads the data into SPSS; some variable names exceed length 8.

The file "atusact_2021.sas" reads the data into SAS.

The file "atusact_2021.do" reads the data into Stata.

If you have SAS, SPSS, or Stata software, open the corresponding program, edit the infile (in SAS), file (in SPSS), or import (in Stata) statement to reference the data file on your computer, and run the program.
